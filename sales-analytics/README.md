# Sales Analytics Project

This project analyses e-commerce sales using:
- Python (Pandas, NumPy)
- MySQL (ETL + data modelling)
- Power BI (KPIs + Dashboard)

## Structure
- data/raw → raw datasets (ignored in Git)
- data/processed → cleaned CSVs
- notebooks → Jupyter/Colab analysis
- sql → MySQL DDL + queries
- powerbi → .pbix dashboards
